# City Explorer - Скопје

Интерактивен веб портал за град Скопје, Македонија.

## Содржина

Проектот содржи 4 главни страници:

1. **Почетна страна (index.html)** - Најава, навигација, популарни места
2. **Галерија (gallery.html)** - Слики со можност за like/dislike и коментирање
3. **Блог/Водич (blog.html)** - Статии и "10 работи кои мора да ги видите во Скопје"
4. **Анкета (survey.html)** - Прашалник за посетителите

## Технологии

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5.3.2
- Bootstrap Icons

## Карактеристики

### Почетна страна
- Стилизиран header и footer
- Навигациско мени
- Модал за најава
- Приказ на популарни места

### Галерија
- Приказ на слики
- Like/Dislike функционалност
- Систем за коментари
- Зачувување на податоци во localStorage

### Блог
- 10 мора-да-се-видат локации во Скопје
- Корисни туристички информации
- Респонзивен дизајн

### Анкета
- Лични податоци (име, е-пошта, возраст)
- Ѕвездичка систем за оценување
- Радио копчиња за избор
- Checkbox опции
- Текстуално поле за коментари
- Валидација на форма

## Како да се користи

1. Отворете `index.html` во веб прелистувач
2. Навигирајте низ различните страници преку менито
3. Испробајте ги интерактивните функции:
   - Најава на почетна страна
   - Like/Dislike и коментирање во галеријата
   - Пополнување на анкетата
